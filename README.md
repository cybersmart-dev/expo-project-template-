
## React native project template 

### complate setup
- Installation
- Run development server
- Build

### Installation

clone git repo

```shell
git clone https://github.com/cybersmart-dev/expo-project-template-.git
```

```shell
cd expo-project-template-
```

```shell

npm install

```

### Run development server

```shell
npx expo start

```
