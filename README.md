
## React native email applicatio ui

### complate setup
- Installation
- Run development server
- Build

### Installation

clone git repo

```shell
git clone https://github.com/cybersmart-dev/React-native-email-application-ui.git
```

```shell
cd React-native-email-application-ui
```

```shell

npm install

```

### Run development server

```shell
npx expo start
```
